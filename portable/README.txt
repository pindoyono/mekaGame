📦 CARA INSTALL:
================================================================================

1. Extract file ZIP ini ke folder yang Anda inginkan
   Contoh: C:\MekaGame\ atau D:\Games\MekaGame\

2. Pastikan struktur folder seperti ini:
   MekaGame-Portable/
   ├── START_MEKAGAME.bat    ← Double-click file ini untuk mulai
   ├── run.py
   ├── out/                  ← Folder aplikasi
   └── README.txt            ← File ini

3. PENTING: Install Python terlebih dahulu
   - Download Python dari: https://www.python.org/downloads/
   - Pilih: Python 3.8 atau lebih baru
   - Saat install, CENTANG "Add Python to PATH"
   - Atau gunakan portable Python (jika disediakan)


🚀 CARA MENJALANKAN:
================================================================================

SANGAT MUDAH - HANYA 1 LANGKAH:

1. Double-click file: START_MEKAGAME.bat
2. Browser akan terbuka otomatis
3. SELESAI! Aplikasi siap digunakan

⚠️  PENTING: JANGAN tutup window command prompt hitam selama menggunakan aplikasi!


💾 DATA & AKUN:
================================================================================

📌 Data Akun & Progress:
   - Tersimpan di browser (localStorage)
   - Otomatis tersimpan setiap kali ada perubahan
   - Data tetap ada meskipun aplikasi ditutup

📌 Backup Data:
   - Buka browser → F12 → Console
   - Ketik: localStorage
   - Copy semua data untuk backup

📌 Multi-User:
   - Setiap siswa buat akun sendiri
   - Bisa bergantian di komputer yang sama
   - Logout dulu sebelum ganti user


🔧 TROUBLESHOOTING:
================================================================================

❌ "Python tidak ditemukan"
   → Install Python dari https://www.python.org/downloads/
   → Atau gunakan portable Python yang disediakan

❌ "Port 3000 sudah digunakan"
   → MekaGame sudah berjalan
   → Atau ada aplikasi lain yang pakai port 3000
   → Tutup aplikasi tersebut dulu

❌ Browser tidak terbuka otomatis
   → Buka manual browser Anda
   → Ketik: http://localhost:3000

❌ Halaman tidak muncul / error 404
   → Pastikan folder 'out' ada dan lengkap
   → Extract ulang file ZIP

❌ Data hilang setelah restart
   → Jangan gunakan mode Incognito/Private
   → Jangan clear browser data/cache
   → Gunakan browser yang sama setiap kali
